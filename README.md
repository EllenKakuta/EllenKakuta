## 💻 Projeto em destaque

### 🔹 Sistema de Controle de Gastos

Sistema desenvolvido em Python com foco em lógica financeira e comportamento do usuário.

**Funcionalidades:**
- Registro de entradas e saídas  
- Controle de despesas fixas  
- Gestão de cartão de crédito  
- Parcelamento de compras  
- Organização de faturas por mês  
- Extrato com filtros  

**Desafio principal:**
Estruturar corretamente a lógica de cartão de crédito, considerando data de compra, fechamento de fatura e distribuição de parcelas ao longo do tempo.
